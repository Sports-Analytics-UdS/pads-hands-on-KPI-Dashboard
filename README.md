# KPI metrics——Dashboard Task (Exemplar)

This repository contains KPI metrics from all big-five league clubs (by Statsbomb).

ℹ️ Due to current technical issues encountered during making request to Statsbomb API, only one season is provided in the exemplar dataset below.

ℹ️ A larger dataset containing five-season KPI data is expected to be uploaded by _9 January, 2025_.

## Technical Glossary

Statsbomb provides the [definition](https://classic.statsbomb.hudl.com/help/sbdata-glossary) of each metric included in this dataset.
